CHAT Visual Identity Guide

Chairs and Tables

Logo

The logo is based on a chair icon created by Hayley Wells on The Noun Project. The opacity change is meant to highlight the shape of both a chair and a table. (Zing!) The font is ADAM.CG PRO by Shrenik Ganatra on Behance. The orange background ("dark") logo should be used for backgrounds that aren't light enough or are too crowded for the light logo. Use your judgement.

Color Palette

Using the orange as a base colour, the palette makes use of a monochromatic colour scheme created through Paletton. We'll stick to using primarily #FF9800 orange and the associated lighter tints, rather than shades #C67600 and #9B5D00. Text overlay should be done in white.

Fonts

The fonts are powered by Google Fonts API in order to be fully functional on the web. Each stylesheet has the necessary link or @import to access Google Fonts. The fonts themselves are Open Sans 800 uppercase 32px for headers, Gentium Basic 400 italic 32px for subheaders, and Gentium Basic 400 18px for body text.

Finding Yourself Lost

Logo

The logo is based on a chair icon created by Hayley Wells on The Noun Project. The opacity change is meant to highlight the shape of both a chair and a table. (Zing!) The font is ADAM.CG PRO by Shrenik Ganatra on Behance. The orange background ("dark") logo should be used for backgrounds that aren't light enough or are too crowded for the light logo. Use your judgement.

Color Palette

The colour palette for FYL derives from #316DAC blue, one of the complement colours from the CHAT palette. FYL also is based on a monochromatic colour scheme created through Paletton. The complement colour, however, is from #FF3900 red-orange, an unofficial official colour of SoundCloud, found somewhere on the web. It's close enough. The red-orange is used as a highlight to tie the blue accents with the SoundCloud embeds.
