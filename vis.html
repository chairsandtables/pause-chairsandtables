CHAT Visual Identity Guide

Chairs and Tables

Logo

The logo is based on a chair icon created by Hayley Wells on The Noun Project. The opacity change is meant to highlight the shape of both a chair and a table. (Zing!) The font is ADAM.CG PRO by Shrenik Ganatra on Behance. The orange background ("dark") logo should be used for backgrounds that aren't light enough or are too crowded for the light logo. Use your judgement.

Color Palette

Using the orange as a base colour, the palette makes use of a monochromatic colour scheme created through Paletton. We'll stick to using primarily #FF9800 and the associated lighter tints, rather than shades #C67600 and #9B5D00. Text overlay should be done in white.
